Introduction

The RAP project aims to enable developers to build rich, AJAX-enabled Web applications by using the Eclipse development model, plug-ins and a Java-only API.

The most recent information regarding RAP can be found on the projects web site. A list of the more noteworthy things available in this build can be found on the RAP New and Noteworthy page.

All documents of the online help assume that you have downloaded and installed the latest RAP build which is available here. We recommend to install the SDK release, so that all steps in the tutorials are doable without limitations.
