About

The Rich Ajax Platform enables developers to build rich, Ajax-enabled Web applications by using the Eclipse development model. User interfaces can be developed using a widget toolkit with SWT API, JFace, and databinding. Applications can be composed of OSGi bundles and also use the well known Eclipse extension points. They even can, but do not have to use the Eclipse Workbench. As an effect, existing RCP applications can be run as Web applications with only minor changes.
