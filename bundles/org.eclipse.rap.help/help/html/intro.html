About

The Rich Ajax Platform (RAP) is a framework for modular, web-based business applications. User interfaces can be developed entirely in Java using a widget toolkit with SWT API, JFace, and databinding. They can, but do not have to use the Eclipse Workbench and Eclipse extension points. Existing RCP applications can be run as Web applications with only minor changes.

Additional Information

  • The most recent version of the RAP Developer Guide can always be found online.
  • Visit the project web site to stay up-to-date on current developments. It also contains further directions to get help in case of problems.
  • We collect answers to frequently asked questions in the RAP FAQ.
  • The RAP Wiki contains information on organizational details of the RAP project, on integration with other technologies, and on specific aspects that are too detailed or technical for the Developer Guide.
